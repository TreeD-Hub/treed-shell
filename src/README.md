# `src`

Основной UI-код TreeD Shell.

Состав:
- `core/` — транспорт, store и командный слой;
- `dashboard/` — конфиг и вычисления главного экрана;
- `ui/` — библиотека ассетов (кнопки/иконки/виджеты);
- `App.tsx` — базовый экран оболочки;
- `config.ts` — чтение режима данных и URL Moonraker.

Инварианты:
- UI-контракт под `960x544`;
- повторяемые UI-элементы создаются через `src/ui`, а не локально в страницах;
- изменение foundation/ui-kit должно каскадно влиять на все страницы, где используются primitives;
- явная индикация online/offline;
- без silent-fail при ошибках транспорта.
